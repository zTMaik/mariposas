# 🦋 Mariposas Interactivas 3D

![Demostración](https://img.shields.io/badge/demo-Three.js-orange)
![Licencia](https://img.shields.io/badge/licencia-CC--BY--NC--4.0-blue)

## 📖 Descripción

Una experiencia interactiva inmersiva que presenta mariposas animadas en un entorno 3D. Utilizando Three.js y técnicas avanzadas de animación, las mariposas vuelan de manera natural mientras el usuario puede explorar el espacio con controles intuitivos.

## ✨ Características

- 🦋 Mariposas 3D con animaciones realistas
- 🎮 Controles de cámara interactivos (arrastrar, zoom, deslizar)
- 🌈 Colores vibrantes y efectos visuales
- 📱 Diseño responsivo para móviles y escritorio
- 🎨 Materiales y texturas procedurales
- 🌟 Efectos de iluminación dinámicos
- 🔄 Animaciones de vuelo naturales

## 🚀 Cómo usar

1. Abre `index.html` en tu navegador web
2. **Arrastrar**: Mantén presionado y mueve el mouse para rotar la vista
3. **Zoom**: Usa la rueda del mouse o pincha en dispositivos táctiles
4. **Deslizar**: En móviles, desliza para navegar por el espacio
5. ¡Observa las mariposas volar libremente!

## 🛠️ Tecnologías utilizadas

- **Three.js**: Motor de renderizado 3D
- **WebGL**: Aceleración gráfica por hardware
- **ES6 Modules**: Estructura de código modular
- **CSS3**: Estilos y layout responsivo
- **HTML5**: Estructura semántica
- **JavaScript**: Lógica de animación e interacción

## 🔧 Características técnicas

- **Geometría procedural**: Alas de mariposa generadas matemáticamente
- **Sistema de partículas**: Movimiento de enjambre natural
- **LOD (Level of Detail)**: Optimización automática
- **Instanced rendering**: Rendimiento optimizado
- **Materials avanzados**: Shaders personalizados
- **Controls responsivos**: Touch y mouse

## 💻 Compatibilidad

- **Escritorio**: Chrome 60+, Firefox 55+, Safari 12+, Edge 79+
- **Móvil**: iOS Safari 12+, Chrome Mobile 70+
- **Hardware**: WebGL compatible
- **Rendimiento**: 30-60 FPS según el dispositivo

- **Boids algorithm**: Comportamiento de enjambre
- **Vertex animation**: Movimiento de alas fluido
